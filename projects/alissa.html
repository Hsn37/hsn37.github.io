Alissa Chatbot

  • Chatbot Application for the Google playstore
  • Framework: React Native
  • Free Version - Google Play
  • Paid Version - Google Play

The chatbot is based on Alice the chatbot, which is a rule based chat engine. It parses an xml script and extracts pairs of questions and answers. The input is matched against those questions and the appropriate answer is chosen. The backend was developed using Python, using a minimal application based on websockets. The paid version has additional themes and uses the text-to-speech feature to give a voice to the agent. The messages that are received are also spoken out loud.
