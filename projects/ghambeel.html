Ghambeel

  • Android Application
  • Framework: Flutter
  • Database: MySQL
  • Ghambeel - Google Play

It is a task management application that allows users to keep track of upcoming tasks and boost their productivity. The application uses various techniques to help users manage their schedule, such as daily reminders, time tracker to observe how much time the user spent on each task, pomodoro timer, and daily motivational quotes to start the day on a high note.
